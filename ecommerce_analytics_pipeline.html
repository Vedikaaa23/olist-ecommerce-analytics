Setup steps — free tier
Step 1 · AWS S3
1.Create S3 bucket: olist-data-raw
2.Download Olist dataset from Kaggle
3.Upload all 9 CSV files to S3
4.Create folders: /orders, /customers, /products
5.Enable versioning on bucket
Cost: Free (5GB tier)
Step 2 · AWS Glue
1.Create Glue Data Catalog
2.Run Glue Crawler on S3 bucket
3.Verify schema auto-detection
4.Write Glue job to join tables
5.Output cleaned Parquet to S3
Cost: 10 DPU-hr/month free
Step 3 · Redshift
1.Create Redshift Serverless workgroup
2.Create tables: orders, customers, products
3.COPY data from S3 using IAM role
4.Run SQL queries in Query Editor v2
5.Create views for KPIs
Cost: 300 RPU-hr free trial
Step 4 · QuickSight
1.Sign up for QuickSight (30-day free)
2.Connect to Redshift as data source
3.Create dataset from views
4.Build revenue, funnel, RFM visuals
5.Publish & screenshot for portfolio
Cost: Free 30-day trial
Redshift table schema
-- Run these in Redshift Query Editor v2 CREATE TABLE orders ( order_id VARCHAR(50) PRIMARY KEY, customer_id VARCHAR(50), order_status VARCHAR(20), order_purchase TIMESTAMP, order_delivered TIMESTAMP, order_estimated TIMESTAMP ); CREATE TABLE order_items ( order_id VARCHAR(50), product_id VARCHAR(50), seller_id VARCHAR(50), price DECIMAL(10,2), freight_value DECIMAL(10,2) ); CREATE TABLE customers ( customer_id VARCHAR(50) PRIMARY KEY, customer_city VARCHAR(100), customer_state VARCHAR(10) ); CREATE TABLE products ( product_id VARCHAR(50) PRIMARY KEY, product_category VARCHAR(100), product_weight_g INT ); -- Load from S3 (replace with your bucket & IAM role ARN) COPY orders FROM 's3://olist-data-raw/orders/olist_orders_dataset.csv' IAM_ROLE 'arn:aws:iam::YOUR_ACCOUNT:role/RedshiftS3Role' CSV IGNOREHEADER 1;
